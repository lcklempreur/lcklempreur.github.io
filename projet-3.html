Projet : Site Web Vitrine

Objectif du projet

Création d'un site web vitrine moderne et responsive pour une petite entreprise locale. Le site devait présenter les services de l'entreprise, inclure une galerie de réalisations et un formulaire de contact fonctionnel.

Technologies et Compétences

  • Frontend : HTML5, CSS3 (avec Flexbox/Grid), JavaScript
  • Design : Conception responsive (Mobile-First)
  • Outils : Visual Studio Code, Git/GitHub
Voir le PDF Télécharger le PDF

Démarche et Résultats

Après une phase de maquettage simple, j'ai développé le site en utilisant des pratiques modernes de développement web. Le JavaScript a été utilisé pour des interactions dynamiques comme la validation du formulaire. Le résultat est un site rapide, accessible et parfaitement consultable sur mobile comme sur ordinateur.
