Projet : Script d'Automatisation PowerShell

Objectif du projet

Le but de ce projet était de développer un script PowerShell pour automatiser la création de comptes utilisateurs dans un environnement Active Directory. Le script devait prendre en entrée un fichier CSV contenant les informations des nouveaux employés (nom, prénom, service, etc.) et créer les comptes, les assigner aux bons groupes de sécurité, et générer un mot de passe temporaire.

Technologies et Compétences Mises en Œuvre

  • Scripting : PowerShell 5.1 / 7
  • Système : Windows Server, Active Directory
  • Format de données : Manipulation de fichiers CSV
  • Logique : Gestion des erreurs, logging des actions

Démarche et Résultats

Le script lit le fichier CSV ligne par ligne. Pour chaque utilisateur, il vérifie si le compte existe déjà. Si non, il crée le compte, définit ses attributs, l'ajoute aux groupes correspondants et génère un rapport final. Ce script a permis de réduire le temps de création de 10 minutes par utilisateur à quelques secondes, tout en éliminant les erreurs de saisie manuelle.
