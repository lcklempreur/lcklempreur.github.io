Projet : Déploiement de GLPI & FusionInventory

Ce projet documente le déploiement complet d'une solution de gestion de parc informatique (ITSM) en utilisant GLPI, enrichie par le plugin FusionInventory pour l'inventaire automatisé. L'ensemble a été installé et configuré sur un serveur Linux (Ubuntu) hébergé dans un environnement virtualisé.

Environnement Technique

  • Hyperviseur : VMware Workstation
  • Système d'exploitation : Ubuntu 23.10 Desktop
  • Serveur Web : Apache2
  • Base de données : MySQL
  • Langage : PHP et ses dépendances
  • Applications : GLPI 10.0.6 & Plugin FusionInventory

Démarche et Résultats

La mise en place a suivi un processus méthodique : préparation de la VM, installation de la pile LAMP (Apache, MySQL, PHP), création de la base de données, déploiement de GLPI, et enfin, intégration de FusionInventory avec configuration des agents sur les postes clients.

Le résultat est une **plateforme ITSM pleinement opérationnelle**, capable de gérer un parc, de suivre les tickets d'assistance et d'automatiser l'inventaire. Ce projet a permis de solidifier mes compétences en administration système Linux, déploiement web et virtualisation.
